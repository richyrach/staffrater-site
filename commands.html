Commands

Everything is slash-commands. Permissions are enforced per command.

Ratings

  • /addadmin — register a staff member (Admin)
  • /removeadmin — remove a staff member (Admin)
  • /setratingchannel — restrict where ratings can happen (Admin)
  • /setresultchannel — channel for new-rating posts (Admin)
  • /postratingui — dropdown UI for rating (Admin)
  • /rate, /editrating, /deleterating
  • /profile, /leaderboard, /exportdata
  • /banfromrating, /unbanfromrating

Tickets

  • /setticketconfig — set category, staff role, log channel (Admin)
  • /postticketbutton — post an “Open Ticket” button (Admin)

Applications

  • /pendingapps — review the next pending application (Admin)
  • /deleteapp — soft-delete an application (Admin)
  • /purgeapp — permanently delete application data (Owner)

Polls

  • /createpoll — options separated by | (Manage Server)
  • /pollresults — results by message id

Moderation & Utilities

  • /warn, /warnings
  • /setmuterole, /mute, /unmute
  • /purge, /kick, /ban, /unban
  • /userinfo, /serverinfo, /avatar
  • /remindme, /todo_add, /todo_list, /todo_done, /todo_del
  • /addquote, /quotes, /quoteid

Owner

  • /servers, /botstats, /sayas, /restart
Tip: You can keep setup minimal — most configuration is done in Discord with slash commands.
