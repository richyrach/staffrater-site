Privacy Policy

This is a plain-language overview. If you need a stricter legal version, we can expand it later.

What data the bot stores

  • Ratings: staff user ID, rater user ID, score (1–5), optional comment, timestamp.
  • Configuration: server ID and configured channels/roles (rating channels, ticket category, etc.).
  • Tickets/applications (if enabled): metadata needed to manage the workflow (channel IDs, status, timestamps).

What data the bot does not collect

  • No reading or storing of your private DMs with the bot (unless you directly submit application answers via the flow).
  • No message content logging across your server as a background process.
  • No selling of user data.

Website login cookies

If you use the dashboard login, the website may set a session cookie (e.g. sr_session) to keep you signed in. This cookie is used only for authentication and is not a tracking cookie.

Data retention

Data is retained while the bot is installed in your server, unless you remove it or request deletion. Server admins can also remove staff from rating lists and delete ratings when needed.

Contact

For privacy questions or deletion requests, contact support via the Discord server: Support
